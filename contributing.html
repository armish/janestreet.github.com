Contributing to Core

To contribute to Core, you first need to sign the Contributor License Agreement.

As a technical matter, here is the way of submitting to core:

  • create a pull-request for the project you want to contribute to (for example: async). The request will be used for discution only, it won't be merged in the public repository in the end because this is not how we work internally.
  • once the change is accepted we will integrated to our development/review process and it will be part of a future release of Core.
